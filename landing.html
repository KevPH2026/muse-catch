Features

不只是笔记，是第二大脑

浏览器、Telegram、API —— 灵感从四面八方来，Muse 帮你接住、提炼、归档。

🔌

全渠道捕获

Chrome 插件一键捕获网页，Telegram Bot 转发消息自动入库，REST API 接入任何工具。Platform-agnostic by design.

🧠

AI 智能提取

粘贴链接或文字，AI 自动生成标题、摘要、关键词、情绪标签。支持 DeepSeek LLM，也内置规则引擎作为后备。

⌨️

快捷键 + 右键菜单

Ctrl+Shift+M 直接捕获当前页面。右键任意位置 →「捕获到 Muse」。灵感闪现时，不需要翻找，不需要等待。

✏️

双击即编辑

Dashboard 卡片墙支持 contenteditable —— 双击标题或摘要直接修改，失焦自动保存。像编辑 Notion 一样丝滑。

📊

灵感看板

卡片式布局，按时间倒序。情绪图标、来源标签、关键词一目了然。灵感不再是散落的碎片，而是可检索的资产。

🔓

开放 API + 数据主权

REST API 是唯一接口。SQLite 本地存储，100% 数据归你。不需要注册、不依赖云服务、不担心关停。

Why Muse?

每天我们消费成千上万条信息。灵感像火花一样闪过，然后消失在信息洪流里。

Muse 的设计哲学很简单：灵感不该丢。浏览器上看到的好句子、Telegram 里聊出的点子、洗澡时冒出的想法 —— 1 秒捕获，AI 整理，永远可查。

不是另一个笔记软件。是你灵感库的操作系统。

Get Started

30 秒安装，1 秒捕获

1

打开扩展页面

Chrome 地址栏输入 chrome://extensions

2

加载扩展

右上角开「开发者模式」→ 点「加载已解压的扩展程序」→ 选 extension 文件夹

3

开捕

浏览任意网页 → 点插件图标或按 Ctrl+Shift+M → 灵感入库 🌀
