Features

Not just notes — your second brain

Ideas come from everywhere. Browser, chat, API — Muse catches them all, processes with AI, stores locally.

🔌

Omnichannel Capture

Chrome extension one-click capture. Any Agent forward-to-save. REST API for any tool. Platform-agnostic by design.

🧠

AI Auto-Extraction

Paste link or text — AI generates title, summary, keywords, emotion tags. DeepSeek V4 Pro + rule engine fallback.

⌨️

Hotkey + Context Menu

Ctrl+Shift+M to capture current page. Right-click anywhere → "Capture to Muse". No app switching.

✏️

Inline Edit

Double-click any title or summary on the card wall — edit in place, auto-save on blur. Fluid as Notion.

📊

Inspiration Dashboard

Card layout, emotion icons, source tags, keywords. Heatmaps, donut charts, bar stats. Ideas as searchable assets.

🔓

Open API + Data Sovereignty

REST API is the only interface. SQLite local storage. 100% your data. No signup. No cloud lock-in.

Why Muse?

Every day we consume thousands of pieces of information. Ideas flash like sparks — then drown in the noise.

Muse's philosophy is simple: ideas shouldn't be lost. A sentence in the browser, a thought in chat, a shower insight — 1 second capture, AI-organized, always searchable.

Not another note app. The operating system for your idea library.
